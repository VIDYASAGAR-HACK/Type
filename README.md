## Description

CyberType is a typing game designed to improve your typing speed and accuracy. It features a cyberpunk aesthetic with digital distortion effects and neon-blue terminal visuals. The game has four difficulty levels, each with increasing complexity.

## Features

-   Four difficulty levels: Easy, Medium, Hard, and Expert.
-   Real-time WPM (Words Per Minute), accuracy, and combo streak tracking.
-   Hacker-themed UI with glitch effects and neon-blue highlights.
-   Sound effects for an immersive experience.

## How to Play

1.  Open the `index.html` file in your browser.
2.  Select a difficulty level.
3.  Start typing the words that appear on the screen.
4.  Try to beat your high score!
